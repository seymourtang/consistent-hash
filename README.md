## Consistent Hash

```go
func main() {
	h := NewHashRing()
	h.JoinMultiNodes(
		"192.168.1.1",
		"192.168.1.2",
		"192.168.1.3",
		"192.168.1.4",
		"192.168.1.5",
		"192.168.1.6")
	h.PrintNodes()
	node := h.GetNode("test.jpeg")
	fmt.Printf("NodeID:%d,NodeIP:%s\n", node.ID, node.IP)
}
```
